This phase tests whether the warm-up actually hands policies into the ordinary PG attraction region. We run exact deterministic tabular updates for a fixed budget, use Meta-MAPG only for the first $K$ steps, and then switch to pure PG. If basin entry is the right story, short warm-up windows should recover a large share of the constant-Meta-MAPG gain.
